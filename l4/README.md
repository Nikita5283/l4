<h1>1.9</h1>

<h2>Формулировка</h2>

Составить программу, которая переворачивает список L, т.е. изменяет ссылки в этом списке так,<br>
чтобы его элементы оказались расположенными в обратном порядке.

<h2>Алгоритм решения</h2>

1. Создать класс Collections
1. Создать поле list для хранения списка эл-тов
1. Создать конструктор для инициализации передаваемого в класс списка
1. Создать метод ReverseList для переворачивания списка

<h4>ReverseList</h4>

1. Создать список такого же размера, как и переданный список
1. Создать переменную ind_rev для записи элементов в перевернутый список равную нулю
1. С помощью for пройти по исходному списку с конца и записывать эл-ты с конца исходного списка в начало перевернутого
