<h1>1.9</h1>

<h2>Формулировка</h2>

Составить программу, которая переворачивает список L, т.е. изменяет ссылки в этом списке так,<br>
чтобы его элементы оказались расположенными в обратном порядке.

<h2>Алгоритм решения</h2>

1. Создать класс Collections.
1. Создать метод ReverseList для переворачивания списка.
1. Создать метод для создания списка с клавиатуры.

<h4>ReverseList</h4>

1. Создать список L методом CreateList.
1. Создать копию списка L.
1. Создать переменную ind_rev для записи элементов в исходный список равную нулю.
1. С помощью for пройти по копии L с конца и записывать эл-ты с конца копии в начало исходного списка.

<h1>2.9</h1>

<h2>Формулировка</h2>

В списке L справа и слева от элемента E вставить элемент F.

<h2>Алгоритм решения</h2>

1. Написать метод для создания списка с клавиатуры.
1. Передавать этому методу эл-т E и индекс на который этот эл-т нужно вставить(indE), введенные с клавиатуры.
1. Когда в цикле для заполнения списка доходим до indE, запускаем цикл в котором перебираем все узлы связного списка, пока не дойдем до узла по нужному индексу и вставляем эл-т E после этого узла.
1. Написать метод для вставки эл-та F(введенного с клавиатуры) справа и слева от эл-та E.
1. Пока не дойдем до indE берем следующий узел.
1. Когда дошли, слева и справа вставляем эл-т F.
