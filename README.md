<h1>5.9</h1>

<h2>Формулировка</h2>

В некотором вузе абитуриенты проходили предварительное тестирование, по результатам<br>
которого они могут быть допущены к сдаче вступительных экзаменов в первом потоке.<br>
Тестирование проводится по трём предметам, по каждому предмету абитуриент может набрать<br>
от 0 до 100 баллов. При этом к сдаче экзаменов в первом потоке допускаются абитуриенты,<br>
набравшие по результатам тестирования не менее 30 баллов по каждому из трёх предметов,<br>
причём сумма баллов должна быть не менее 140. На вход программы подаются сведения о<br>
результатах предварительного тестирования. Известно, что общее количество участников<br>
тестирования не превосходит 500.<br>
В первой строке вводится количество абитуриентов, принимавших участие в тестировании, N.<br>
Далее следуют N строк, имеющих следующий формат:<br>
<Фамилия><Имя><Баллы><br>
Здесь <Фамилия> – строка, состоящая не более чем из 20 символов; <Имя> – строка, состоящая не<br>
более чем из 15 символов, <Баллы> – строка, содержащая два целых числа, разделенных<br>
пробелом – баллы, полученные на тестировании по каждому из трёх предметов. При этом<br>
<Фамилия> и <Имя>, <Имя> и <Баллы> разделены одним пробелом. Пример входной строки:<br>
Романов Вельямин 48 39 55<br>
Напишите программу, которая будет выводить на экран фамилии и имена абитуриентов,<br>
допущенных к сдаче экзаменов в первом потоке. При этом фамилии должны выводиться в<br>
алфавитном порядке.

<h2>Алгоритм решения</h2>

1. Создать метод для записи данных с клавиатуры в файл
1. Создать метод для чтения данных из файла и работы с ними
1. Записать в сортированный список(SortedList) с помощью split и join ключи - фамилии и имена, значения - баллы
1. Проверить, что по каждому тестированию набрано не менее 30 баллов и записывать каждый балл в сумму
1. Если цикл не заканчивается с помощью break проверить сумму и вывести на экран ключ из списка(фамилию и имя)

-- уменьшить координаты x и y на 1;<br>

– Point p, целое число (левосторонняя операция, уменьшается координата х);<br>
– целое число, Point p (правосторонняя операция, уменьшается координата y);<br>
– Point p – вычисляется расстояние до точки p, результатом должно быть вещественное число.
